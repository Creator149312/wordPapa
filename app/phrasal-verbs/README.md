# Phrasal Verbs Module

## 📋 Overview
Reference module for English phrasal verbs with definitions, examples, and usage context.

## 🎯 Purpose
- Provide comprehensive phrasal verb reference
- Display phrasal verb definitions and meanings
- Show usage examples and context
- Explain verb + particle combinations
- Support vocabulary learning for phrasal verbs
- Generate SEO-friendly phrasal verb pages

## 📁 File Structure
- **page.js** - Main phrasal verbs reference page
- **PHRASALVERBS.js** - Phrasal verbs data/definitions

## 🔗 Routes
- `/phrasal-verbs` - Main phrasal verbs reference page

## 📦 Dependencies
- Phrasal verb definition data
- React components for display
- Styling for readable layout

## 🔄 Related Modules
- `/define` - General word definitions
- `/browse` - Word discovery
- `/thesaurus` - Synonym lookup

## 🚀 Key Features
1. **Phrasal Verb Listing** - Comprehensive phrasal verb list
2. **Definitions** - Clear meanings for each verb
3. **Examples** - Usage examples in context
4. **Categories** - Grouped by type or category
5. **Search** - Find specific phrasal verbs
6. **Sorting** - Sort by difficulty or alphabetically

## 📊 Content Structure
Each phrasal verb entry includes:
- Base verb + particle (e.g., "put up")
- Definition
- Example sentences
- Synonyms
- Related phrasal verbs
- Difficulty level
